# Rua-quanter

<img src="img/yyf_raw.jpg" height="200" align="right"/>

Rua-quanter是应某神秘组织邀请编写的股票筛选工具，实现了简单策略。

## 环境部署

建议使用Anaconda环境（Python 3.x）：

```bash
conda create -n quanter
source activate quanter
pip install tushare -i https://pypi.tuna.tsinghua.edu.cn/simple
```

此后每次`source activate quanter`切换入conda虚拟环境即可正常使用。
